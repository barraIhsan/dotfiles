my arch linux dotfiles, managed with [chezmoi](https://www.chezmoi.io/).  
I also "extend" it with [chezmoi_modify_manager](https://github.com/VorpalBlade/chezmoi_modify_manager/) (ini-like "patch"[^1]), [jq](https://github.com/jqlang/jq) (json "patch"[^1]), and also ["rootmoi"](https://github.com/twpayne/chezmoi/discussions/1510#discussioncomment-7808924)

oh what wm/de do I use? I'm a simple man, so KDE Plasma. I'm not really accustomed to wm's workflow (split/one window(s) per workspace and switch between them)

## Setup

1. Install [chezmoi](https://www.chezmoi.io/install/), [chezmoi_modify_manager](https://vorpalblade.github.io/chezmoi_modify_manager/installation.html#:~:text=Do,these) and [jq](https://jqlang.org/download/) if you haven't installed them yet
2. Clone it with `chezmoi init https://github.com/barraIhsan/dotfiles`
3. Apply it with `chezmoi apply`, you can also see what'll change by using `chezmoi diff` too
